# My Shop

Це простий інтернет-магазин, створений на HTML, CSS та JavaScript.

## Функції:
- Перегляд товарів
- Кошик з анімацією
- Модальне оформлення замовлення
- Інтеграція з LiqPay (у процесі)
- Надсилання email (у процесі)
